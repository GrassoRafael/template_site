# 🌐 Projeto Web – Site Responsivo com HTML e Bootstrap

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um site responsivo utilizando HTML, CSS e Bootstrap, com foco em organização de layout, usabilidade e adaptação para diferentes dispositivos.

O objetivo é demonstrar na prática a construção de interfaces modernas para web, aplicando boas práticas de desenvolvimento front-end.

---

## 🚀 Funcionalidades

- 📱 Layout totalmente responsivo (mobile, tablet e desktop)
- 🎨 Interface moderna utilizando Bootstrap
- 🧭 Navegação estruturada e intuitiva
- 🧩 Componentes reutilizáveis (cards, navbar, botões)
- ⚡ Carregamento leve e rápido

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
